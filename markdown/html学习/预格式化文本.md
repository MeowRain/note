# 预格式化文本

```html

<pre>
    床前明月光，疑是地上霜
    举头望明月，低头思故乡
</pre>
<!-- 上面即为预格式化文本 -->
<!-- 预格式化文本可以显示空格，比如下面 -->

        静夜思 李白

    举头望明月，低头思故乡
</pre>
```
